O que estou estudando atualmente

Sistemas Digitais

Sistemas Digitais é a disciplina que estuda a base fundamental de toda a computação moderna. Enquanto em outras matérias você aprende a "conversar" com o computador (através de linguagens de programação como Python ou Java), em Sistemas Digitais você aprende como o computador "pensa" no nível mais básico. O foco é entender como a informação é representada e processada usando apenas dois estados: ligado ou desligado (1 ou 0).

Web Design

Web Design é a disciplina que estuda a base de como construímos a interação humana com a internet. Enquanto em outras matérias você aprende a criar a 'lógica' do servidor (o back-end), em Web Design você aprende a construir a 'interface' visual que o usuário final realmente vê e utiliza. O foco é entender como usar as tecnologias fundamentais (HTML, CSS e JavaScript) para estruturar conteúdo, aplicar estilo e criar interatividade de forma acessível e funcional.

Programação de Computadores

Programação de Computadores é a disciplina que ensina a arte de "dar instruções" a uma máquina para que ela execute tarefas. Enquanto matérias como Sistemas Digitais explicam como o computador funciona por dentro, esta disciplina ensina como fazê-lo trabalhar para você, usando uma linguagem de programação como ponte de comunicação. O foco é utilizar Python para dominar os conceitos universais da lógica de programação: como armazenar dados (variáveis), tomar decisões (condicionais) e repetir ações (laços), capacitando você a escrever seus primeiros algoritgos e solucionar problemas reais.
